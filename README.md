Installation
------------
###Arch
Install `bdf-tewi-git` from the [AUR][aur page].
[aur page]: https://aur.archlinux.org/packages/bdf-tewi-git/ "AUR page"
###Others
1. Clone the repo
2. Copy fonts to `~/.fonts` (or any subdirectory of it, create `~/.fonts` if needed)
3. Do `fc-cache`
